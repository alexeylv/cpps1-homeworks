# Задание 1. Таблица истинности

Напишите программу, которая выводит таблицу истинности для логических операторов `||`, `&&`, `^`.

## Результат работы программы:

```
Оператор: ||
true    true    true
false   true    true
true    false   true
false   false   false

Оператор: &&
true    true    true
false   true    false
true    false   false
false   false   false

Оператор: ^
true    true    false
false   true    true
true    false   true
false   false   false
```
